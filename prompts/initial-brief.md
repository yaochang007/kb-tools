# Initial Project Brief

Use this prompt when starting a new project from this template.

```md
You are helping start a new language-neutral project.

Project name:

Project purpose:

Primary users:

Problem to solve:

Expected inputs:

Expected outputs:

Constraints:

Known integrations:

Preferred stack, if any:

Unknowns to resolve first:

Please read `AGENTS.md`, `README.md`, `docs/architecture.md`,
`docs/decisions.md`, and `tasks/todo.md` before proposing or making changes.
Keep edits scoped, preserve existing content, do not use sudo, and keep secrets
out of code, logs, prompts, and docs.
```

## Notes

- Replace blank fields with project-specific details.
- Add links to reference material when they are safe to share.
- Move durable decisions into `docs/decisions.md`.
- Move actionable work into `tasks/todo.md`.
